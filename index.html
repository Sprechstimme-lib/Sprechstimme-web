Advanced Synthesis Techniques

Professional sound design tools at your fingertips

Basic Waveforms

Classic oscillator types including sine, square, sawtooth, triangle, and pulse waves with variable duty cycle.

sp.waves.sine | sp.waves.square | sp.waves.sawtooth

FM Synthesis

Frequency modulation synthesis for complex, evolving timbres. Control carrier and modulator frequencies independently.

sp.waves.fm(carrier_freq, mod_freq, mod_depth)

Wavetable Synthesis

Morph between custom wavetables for dynamic timbral evolution. Create complex sounds from simple building blocks.

sp.waves.wavetable(tables, position)

Noise Generators

White, pink, brown, and blue noise for percussion, ambience, and sound effects. Full spectral control.

sp.waves.white_noise | sp.waves.pink_noise

Audio Analysis Tools

Understand and visualize your audio with professional analysis capabilities

Pitch Detection

Multiple algorithms including autocorrelation, YIN, and HPS for accurate pitch detection across different audio types.

pitch = sp.analysis.detect_pitch(audio)
confidence = sp.analysis.pitch_confidence(audio)

FFT Spectrum Analysis

Fast Fourier Transform for frequency domain analysis. Visualize harmonic content and spectral characteristics.

spectrum = sp.analysis.fft(audio)
frequencies, magnitudes = sp.analysis.spectrum()

Onset Detection

Detect note onsets and transients for rhythmic analysis. Essential for beat tracking and tempo-synced effects.

onsets = sp.analysis.detect_onsets(audio)
timing = sp.analysis.onset_times()

Tempo Estimation

Automatic BPM detection for rhythmic audio. Synchronize your compositions with existing recordings.

bpm = sp.analysis.detect_tempo(audio)
beats = sp.analysis.beat_positions()

Advanced Synthesis

Intermediate
Python 
import sprechstimme as sp
from functools import partial

# Create a custom synth with FM synthesis
sp.new('lead')
sp.create(
    'lead',
    wavetype=sp.waves.fm,
    filters=[
        partial(sp.filters.moog_ladder, cutoff=2000, resonance=0.7),
        partial(sp.filters.chorus, depth=0.3)
    ],
    envelope={
        'attack': 0.02,
        'decay': 0.1,
        'sustain': 0.6,
        'release': 0.4
    }
)

# Play with the custom synth
sp.play('lead', ['C4', 'E4', 'G4'], duration=2.0)

Multi-Track Composition

Advanced
Python 
import sprechstimme as sp
from functools import partial

# Create instruments
sp.new('bass')
sp.create('bass', wavetype=sp.waves.sawtooth,
    filters=[partial(sp.filters.low_pass, cutoff=200)])

sp.new('lead')
sp.create('lead', wavetype=sp.waves.square)

sp.new('pad')
sp.create('pad', wavetype=sp.waves.sine)

# Create a multi-track song
song = sp.Song(bpm=120)

# Bass line
for beat in range(0, 16, 2):
    song.add('bass', 'bass', 'C2', beat_position=beat, duration=1.5)

# Lead melody
melody = ['C4', 'E4', 'G4', 'E4', 'F4', 'A4', 'C5', 'A4']
for i, note in enumerate(melody):
    song.add('melody', 'lead', note, beat_position=i*2, duration=1)

# Pad chords
song.add_chord('pad', 'pad', 'C4', beat_position=0, duration=8)
song.add_chord('pad', 'pad', 'F4', beat_position=8, duration=8)

# Export the composition
song.export('my_composition.wav')

Audio Analysis

Intermediate
Python 
import sprechstimme as sp
import numpy as np

# Generate some audio
sp.new('synth')
sp.create('synth', wavetype=sp.waves.sawtooth)
audio = sp.generate('synth', 'A4', duration=2.0)

# Analyze the audio
pitch, confidence = sp.analysis.detect_pitch(audio)
print(f'Detected pitch: {pitch:.2f} Hz (confidence: {confidence:.2%})')

# Get frequency spectrum
spectrum = sp.analysis.fft(audio)
frequencies, magnitudes = sp.analysis.get_spectrum(spectrum)

# Find dominant frequency
dominant_freq = frequencies[np.argmax(magnitudes)]
print(f'Dominant frequency: {dominant_freq:.2f} Hz')

# Detect tempo (if rhythmic)
bpm = sp.analysis.detect_tempo(audio)
print(f'Estimated tempo: {bpm:.1f} BPM')

Custom DSP Function

Advanced
Python 
import numpy as np
import sprechstimme as sp
from functools import partial

# Create a custom tremolo effect
def tremolo(signal, sample_rate, rate=5, depth=0.5):
    '''
    Tremolo: amplitude modulation effect
    rate: modulation frequency in Hz
    depth: modulation depth (0-1)
    '''
    t = np.arange(len(signal)) / sample_rate
    modulator = 1 + depth * np.sin(2 * np.pi * rate * t)
    return signal * modulator

# Create a custom waveform with harmonics
def rich_wave(t, freq=440, amp=1.0):
    '''
    Custom waveform with multiple harmonics
    '''
    fundamental = np.sin(2 * np.pi * freq * t)
    harmonic2 = 0.5 * np.sin(4 * np.pi * freq * t)
    harmonic3 = 0.3 * np.sin(6 * np.pi * freq * t)
    harmonic5 = 0.2 * np.sin(10 * np.pi * freq * t)
    return amp * (fundamental + harmonic2 + harmonic3 + harmonic5) / 2

# Use custom functions
sp.new('custom')
sp.create(
    'custom',
    wavetype=rich_wave,
    filters=[partial(tremolo, rate=4, depth=0.4)]
)

sp.play('custom', 'A4', duration=3.0)
