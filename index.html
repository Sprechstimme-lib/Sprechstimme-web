Sprechstimme

A modular Python synthesizer and sequencer for creating
music programmatically

and compose musical sequences with just a few lines of code.

Complete API reference for the Sprechstimme library

Core Functions

new(name)

Register a new synthesizer with default settings.

Parameters:
  • name (str): Name identifier for the synth
Python 
import sprechstimme as sp

sp.new('lead')

create(name, wavetype=None, filters=None, envelope=None, poly=True)

Configure a synthesizer with specific parameters including waveforms, filters, ADSR envelope, and polyphony.

Parameters:
  • name (str): Name of the synth to configure
  • wavetype (callable or str, optional): Waveform generator function or preset ("sine", "square", "sawtooth", "triangle", "noise") or custom function
  • filters (list, optional): List of filter functions (use functools.partial for parameters)
  • envelope (dict, optional): ADSR envelope {"attack": float, "decay": float, "sustain": float, "release": float}
  • poly (bool, optional): Enable polyphony. Default: True
Python 
from functools import partial

sp.create('bass',
    wavetype=sp.waves.sawtooth,
    filters=[partial(sp.filters.low_pass, cutoff=800)],
    envelope={'attack': 0.01, 'decay': 0.1, 'sustain': 0.7, 'release': 0.2}
)

play(name, notes, duration=0.5, sample_rate=44100)

Play notes on a synthesizer. Accepts MIDI numbers, frequencies (Hz), note names ("C4"), or lists for chords.

Parameters:
  • name (str): Synth name
  • notes (int/float/str/list): Single note, list of notes (chord), or list of (note, duration) tuples (sequence)
  • duration (float, optional): Duration in seconds. Default: 0.5
  • sample_rate (int, optional): Sample rate in Hz. Default: 44100
Python 
# Single note
sp.play('lead', 'C4', duration=0.5)

# Chord
sp.play('lead', [60, 64, 67], duration=1.0)

# MIDI number
sp.play('lead', 60, duration=0.5)

get(name)

Get configuration summary for a synthesizer.

Parameters:
  • name (str): Name of the synth
Returns:

Dictionary with synth configuration

Python 
config = sp.get('lead')
print(config)

Utility Functions

midi_to_freq(midi_note)

Convert MIDI note number to frequency in Hz.

Parameters:
  • midi_note (int): MIDI note number (0-127)
Returns:

float: Frequency in Hz

Python 
freq = sp.midi_to_freq(69)  # A4 = 440 Hz

note_to_freq(note)

Convert note name to frequency. Supports sharps (#) and flats (b).

Parameters:
  • note (str): Note name (e.g., "C4", "A#3", "Bb2")
Returns:

float: Frequency in Hz

Python 
freq = sp.note_to_freq('A4')  # 440 Hz
freq2 = sp.note_to_freq('C#4')

Track Class

Track(bpm=120, sample_rate=44100)

Create a new musical track/sequence with BPM-based timing.

Parameters:
  • bpm (int, optional): Beats per minute. Default: 120
  • sample_rate (int, optional): Sample rate in Hz. Default: 44100
Python 
track = sp.Track(bpm=128)

Track.add(synth, notes, duration=1)

Add an event to the track.

Parameters:
  • synth (str): Name of synth to use
  • notes (int/float/str/list): Note(s) to play
  • duration (float, optional): Duration in beats. Default: 1
Python 
track.add('lead', 'C4', duration=1)
track.add('bass', [60, 64, 67], duration=2)

Track.addChord(synth, chord, duration=1)

Add a chord using chord notation. Supports major (e.g., "C4") and minor chords (e.g., "Am3").

Parameters:
  • synth (str): Name of synth to use
  • chord (str): Chord notation (e.g., "C4", "Am3", "F#5")
  • duration (float, optional): Duration in beats. Default: 1
Python 
track.addChord('pad', 'C4', duration=2)  # C major
track.addChord('pad', 'Am3', duration=2)  # A minor

Track.play()

Play the track through audio output.

Python 
track.play()

Track.export(filename="output.wav")

Export track to a WAV file.

Parameters:
  • filename (str, optional): Output filename. Default: "output.wav"
Python 
track.export('my_track.wav')

Song Class

Song(bpm=120, sample_rate=44100)

Create a multi-track song with universal beat counter. Supports precise beat positioning including fractional beats (e.g., 200.5).

Parameters:
  • bpm (int, optional): Beats per minute for the entire song. Default: 120
  • sample_rate (int, optional): Sample rate in Hz. Default: 44100
Python 
song = sp.Song(bpm=140)

Song.add(track_name, synth, notes, beat_position, duration=1)

Add an event to a specific track at a specific beat position. Track is auto-created if it doesn't exist.

Parameters:
  • track_name (str): Track identifier (auto-created if new)
  • synth (str): Name of synthesizer to use
  • notes (int/float/str/list): Note(s) to play
  • beat_position (float): Start beat (supports fractional beats like 200.5)
  • duration (float, optional): Duration in beats. Default: 1
Python 
song.add('melody', 'lead', 'C4', beat_position=0, duration=1)
song.add('melody', 'lead', 'E4', beat_position=1.5, duration=0.5)

Song.add_chord(track_name, synth, chord, beat_position, duration=1)

Add a chord event to a specific track at a specific beat position.

Parameters:
  • track_name (str): Track identifier
  • synth (str): Name of synthesizer to use
  • chord (str): Chord notation (e.g., "C4", "Am3")
  • beat_position (float): Start beat
  • duration (float, optional): Duration in beats. Default: 1
Python 
song.add_chord('chords', 'pad', 'C4', beat_position=0, duration=4)

Song.play()

Render and play the entire song through audio output.

Python 
song.play()

Song.export(filename="output.wav")

Render and export the song to a WAV file.

Parameters:
  • filename (str, optional): Output filename. Default: "output.wav"
Python 
song.export('my_song.wav')

Song.get_duration()

Get the total duration of the song.

Returns:

Dictionary with 'beats', 'seconds', and 'formatted' time

Python 
duration = song.get_duration()
print(duration)  # {'beats': 4, 'seconds': 2.0, 'formatted': '0:02.00'}

Song.list_tracks()

Get information about all tracks in the song.

Returns:

Dictionary with track names, event counts, and synths used

Python 
tracks = song.list_tracks()
print(tracks)

Waveform Generators

sp.waves.*

Built-in waveform generators. All have signature: func(t, freq=440, amp=1.0)

Available Waveforms:
  • sp.waves.sine - Pure sine wave
  • sp.waves.square - Square wave
  • sp.waves.sawtooth - Sawtooth wave
  • sp.waves.triangle - Triangle wave
  • sp.waves.noise - White noise
Python 
sp.create('synth1', wavetype=sp.waves.sine)
sp.create('synth2', wavetype=sp.waves.square)
sp.create('synth3', wavetype=sp.waves.sawtooth)

Custom Waveform Functions

Create your own custom waveform generator functions. Must accept parameters: t (time array), freq (frequency), and amp (amplitude).

Python 
import numpy as np

def custom_wave(t, freq=440, amp=1.0):
    # Create a custom waveform (example: sine + sine harmonic)
    return amp * (np.sin(2 * np.pi * freq * t) +
                  0.5 * np.sin(4 * np.pi * freq * t))

sp.create('custom', wavetype=custom_wave)

Audio Filters

sp.filters.low_pass(signal, sample_rate, cutoff=1000.0, order=4)

Butterworth low-pass filter. Use with functools.partial to set parameters.

Parameters:
  • signal (array): Audio signal
  • sample_rate (int): Sample rate
  • cutoff (float, optional): Cutoff frequency in Hz. Default: 1000.0
  • order (int, optional): Filter order. Default: 4
Python 
from functools import partial

sp.create('bass', filters=[partial(sp.filters.low_pass, cutoff=500)])

sp.filters.high_pass(signal, sample_rate, cutoff=200.0, order=4)

Butterworth high-pass filter.

Parameters:
  • cutoff (float, optional): Cutoff frequency in Hz. Default: 200.0
  • order (int, optional): Filter order. Default: 4

sp.filters.band_pass(signal, sample_rate, low=300.0, high=3000.0, order=4)

Butterworth band-pass filter.

Parameters:
  • low (float, optional): Low cutoff in Hz. Default: 300.0
  • high (float, optional): High cutoff in Hz. Default: 3000.0
  • order (int, optional): Filter order. Default: 4

sp.filters.echo(signal, sample_rate, delay=0.25, decay=0.4)

Echo/delay effect.

Parameters:
  • delay (float, optional): Delay time in seconds. Default: 0.25
  • decay (float, optional): Decay factor (0-1). Default: 0.4
Python 
from functools import partial

sp.create('lead', filters=[partial(sp.filters.echo, delay=0.5, decay=0.6)])

sp.filters.simple_distortion(signal, sample_rate, gain=1.0, threshold=0.8)

Clipping-based distortion effect.

Parameters:
  • gain (float, optional): Gain multiplier. Default: 1.0
  • threshold (float, optional): Clipping threshold. Default: 0.8

Custom Filter Functions

Create your own custom filter functions. Must accept parameters: signal (audio array) and sample_rate.

Python 
from functools import partial
import numpy as np

def custom_filter(signal, sample_rate, amount=0.5):
    # Example: simple tremolo effect
    t = np.arange(len(signal)) / sample_rate
    modulator = 1 + amount * np.sin(2 * np.pi * 5 * t)  # 5 Hz
    return signal * modulator

sp.create('synth', filters=[partial(custom_filter, amount=0.3)])

Examples

Quick Start

Python 
import sprechstimme as sp

# Create and configure a synthesizer
sp.new('lead')
sp.create('lead', wavetype=sp.waves.sawtooth)

# Play a single note (MIDI number 60 = middle C)
sp.play('lead', 60, duration=0.5)

# Play using note names
sp.play('lead', 'C4', duration=0.5)

# Play a chord
sp.play('lead', [60, 64, 67], duration=1.0)  # C major chord

Using Filters and Envelopes

Python 
import sprechstimme as sp
from functools import partial

# Create a synth with filters
sp.new('bass')
sp.create(
    'bass',
    wavetype=sp.waves.sawtooth,
    filters=[
        partial(sp.filters.low_pass, cutoff=800),
        partial(sp.filters.echo, delay=0.3, decay=0.5)
    ],
    envelope={
        'attack': 0.01,
        'decay': 0.1,
        'sustain': 0.7,
        'release': 0.2
    }
)

sp.play('bass', 'A2', duration=1.0)

Track Sequencing

Python 
import sprechstimme as sp

# Create synthesizers
sp.new('lead')
sp.create('lead', wavetype=sp.waves.square)
sp.new('bass')
sp.create('bass', wavetype=sp.waves.sawtooth)

# Create a track at 120 BPM
track = sp.Track(bpm=120)

# Add notes (duration in beats)
track.add('bass', notes='C2', duration=4)
track.add('lead', notes='C4', duration=1)
track.add('lead', notes='E4', duration=1)
track.add('lead', notes='G4', duration=1)
track.add('lead', notes='C5', duration=1)

# Play the sequence
track.play()
track.export('my_composition.wav')

Multi-Track Song with Beat Positioning

Python 
import sprechstimme as sp
from functools import partial

# Create synthesizers
sp.new('bass')
sp.create('bass', wavetype=sp.waves.sawtooth,
    filters=[partial(sp.filters.low_pass, cutoff=200)])

sp.new('lead')
sp.create('lead', wavetype=sp.waves.square)

# Create a song at 120 BPM
song = sp.Song(bpm=120)

# Add bass notes at specific beat positions
song.add('bass_track', 'bass', 'C2', beat_position=0, duration=2)
song.add('bass_track', 'bass', 'G1', beat_position=2, duration=2)

# Add melody with half-beat precision
song.add('melody', 'lead', 'C4', beat_position=1, duration=0.5)
song.add('melody', 'lead', 'E4', beat_position=1.5, duration=0.5)
song.add('melody', 'lead', 'G4', beat_position=2, duration=1)

# Add chords
song.add_chord('chords', 'lead', 'C4', beat_position=0, duration=4)

song.play()
song.export('my_song.wav')
